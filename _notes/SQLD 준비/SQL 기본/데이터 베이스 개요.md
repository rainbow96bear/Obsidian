# 테이블 용어

- **테이블** : 데이터의 저장 장소로 데이터베이스의 가장 기본적인 개념
- **행** : 로우, 튜플, 인스턴스 라고도 하며 가로 방향으로 이루어진 연결된 데이터
- **열** : 컬럼이라고도 하며 세로 방향으로 이루어진 특정 속성
- **기본 키** : 행을 특정할 수 있도록 하는 한 개 이상의 칼럼
- **외부 키** : 다른 테이블에서 기본 키로 사용되고 있는 키
