

## 폴더
### controller
사용자의 요청을 처리한 후 뷰에 모델 객체를 넘겨주는 역할
### service
controller에 의하여 호출되고 사용자의 요구사항을 처리하는 역할
### repository
service에 의하여 호출되고 DB의 CRUD 작업
### DTO
데이터를 전송하기 위한 객체
