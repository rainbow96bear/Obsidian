# Sui란?

# Sui Move
Diem에서 만들었던 언어를 오픈소스 사용성에 맞게 수정한 것   
동적 디스패치를 지원하지 않아 재진입이 불가능하다는 특징을 가지고 있습니다.

Object가 소유되었는지, 공유되었는지, 가변적인지, 불변적인지 명확히 나타내기에 Move 기반의 다른 Layer1보다 깔끔합니다.
